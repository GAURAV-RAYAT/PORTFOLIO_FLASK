## 🚀 About The Project

This is my personal portfolio website built with **Flask (Python)** and deployed on **Vercel**. It serves as a central hub to showcase my journey in **Mathematics** and **Data Science**, featuring my projects, resume, and real-time updates.

### ✨ Key Features
- **🤖 AI Chatbot:** A custom-trained assistant (powered by OpenRouter/GPT-3.5) that answers questions about my resume, projects, and skills.
- **👀 Live View Counter:** Real-time visitor tracking using a custom API integration.
- **🔗 Dynamic LinkedIn Feed:** A curated section displaying my latest professional updates and achievements.
- **📧 Contact System:** Fully functional contact form integrated with `Flask-Mail` for direct emails.
- **📱 Responsive Design:** A modern, dark-themed UI that works perfectly on mobile and desktop.
